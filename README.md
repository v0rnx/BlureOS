# BlureOS

Данная ос создаёться для обхода всех возможных ограничений и блокировок таких как РКН и Суверенный интернет.

Ос не будет иметь ни каких ограничений и будут использоватся свои сервисы и приложения, а также будет доступ к зарубежным сервисам.

Данная ос создается пока что исключительно для google pixel 6 pro EU. Тесты не советуются на других устройствах. Все риски вы берёте исключительно на себя.
